# fallback font

You also have to account for users who may not be able to access the Google Fonts.

```css
font-family: "Work Sans", "Arial", sans-serif;
font-family: "Abril Fatface", cursive, serif;
```

- "Work Sans": google font
- "Arial": fallback font
- cursive: google font
- serif: fallback font
