# figcaption

- an element used to describe the media in the `<figure>` tag
- similar to `p` but only go inside `<figure>`

```html
<figure>
  <img src="overwatch.jpg" />
  <figcaption>This picture shows characters from Overwatch.</figcaption>
</figure>
```
