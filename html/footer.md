# footer

- `<head>` / `<body>`
- `<header>` / `<main>` / `<footer>`

Q: Do we put the `<footer>` inside `<body>`?
A: The `<header>` / `<main>` / `<footer>` semantic refer to the editorial content structure which, as a whole, is represented in the document `<body>`
