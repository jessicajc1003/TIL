# Python Comments

- Comments starts with a `#`, and Python will ignore them:
```py
#This is a comment
print("Hello, World!")
```

- Multi Line Comments
```py
#This is a comment
#written in
#more than just one line
print("Hello, World!")
```

- how to comment out elements?
1. highlights the element or put the cursor at the end 
2. press `Ctrl + /` 

- undo the comments: `Ctrl + z`
