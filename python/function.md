## index()
`index()` - when you want to find the index number of the list item

```py
x = [1, 2, 3, 4]

print(x.index(3))
```

## in 
When you just want if the list item is inside the list regardless of its position

```py
y = ["hello", "there"]

print("hello" in y)

#True
```
