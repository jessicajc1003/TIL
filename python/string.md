# string 

```py
print("print("what to print")") #syntax error due to using double quotes twice

print("print('what to print')") #use either double quote or single quote in one element
```

# f-string
```py
>>> name = "Eric"
>>> age = 74
>>> f"Hello, {name}. You are {age}."
'Hello, Eric. You are 74.'
```
