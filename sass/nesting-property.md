# Nesting property

Nest common CSS properties by using a `:` colon suffix after the name of the property

```scss
.parent {
  font : {
    family: Roboto, sans-serif;
    size: 12px;
    decoration: none;
  }
}

// this is same as following in css
.parent {
  font-family: Roboto, sans-serif;
  font-size: 12px;
  font-decoration: none;
}
```
